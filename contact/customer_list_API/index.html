API Endpoints for Customer List


The following are the available routes

Root Route: Displays available routes.

GET /api/customers: Returns all customers.

GET /api/customers/1: Fetches a specific customer by ID.

GET api/customers/8/Valdez/a@email.com/20/Porac/Student: Retrieves and displays parameters.

POST /api/customers: Adds a new customer through form submission.

DELETE /api/delete/1: Deletes a customer by ID.
